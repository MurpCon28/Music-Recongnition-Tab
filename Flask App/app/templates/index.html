Music Recognition Tab 2.1

Information

This application uses the microphone and listens to a song for 20 seconds, the you can download the file and go to the uploads section and upload the file and the application will send the file to the model and search the database for a match.

Along with the song result, links from YouTube and UltimateGuitar Tab are displayed so the user can learn the song on guitar.

The application database only contains a few songs to test the prediction, they are Jailbreak by Thin Lizy, City of Blinding Lights by U2, Hey by Pixies, Monkey Man by The Rolling Stones, Hey Jude by The Beatles and Everlong by Foo Fighters. It also has Panam by Van Halen but it can not find a match for it, it stead says one of the other songs is the match.
